Mrp7Pred

Mrp7Pred is an online server for discovering novel MRP7 modulators. For more details, please check our PUBLICATION.

Start Help

What is MRP7

  • MRP7, or ABCC10, is the 10th member of ABCC subfamily, responsible for the transportation of various endo- and xeno-biotics across the cytoplasm membrane.
  • MRP7 substrates include Leukotriene C4, E217βG, paclitaxel, Vinca alkaloids, gemcitabine, docetaxel, epothilone B

    • Left: Predicted topological structure of MRP7. Right: An MRP7 homology model built in our lab. TMD: transmembrane domain; NBD: nucleotide-binding domain; Lasso: lasso motif.

    Instructions

    Data Preparation

    Mrp7Pred takes SMILES as inputs. The users should prepare their list of compounds in a csv format which has two columns "name" and "smiles". For sample data see examples.

    Making Predictions

    Automatic featurization and prediction. Featurization might fail if your SMILES string contains disconnected structures. Results will be displayed once the prediction is done.

    * Execution time varies by structural complexity.

    Getting Results

    Results in table format will be generated automatically with scores. Compounds with score > 0.5 is considered an MRP7 modulator.
